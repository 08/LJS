LoginViewControllerDelegate Protocol Reference

Declared in LoginViewController.h

Overview

The LoginViewControllerDelegate has four responsibilities:

  1. dismissing the LoginViewController from the parent view
  2. collecting information about the login attempt
  3. providing an authentication and a protected request
  4. providing the username and password as a properly formatted JSON object for the domain
