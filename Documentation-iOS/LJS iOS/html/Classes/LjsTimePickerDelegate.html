Overview

On iOS devices the Date and Time format can be overridden by the user settings, so regardless of Locale and NSDateFormatter settings you can end up with unexpected/unparseable data and time values. This can cause major headaches when trying to layout views that contain date and time references.

LjsTimePickerDelegate solves this problem by bypassing the normal Foundation date formatting and replacing it with LjsDateHelper methods.
