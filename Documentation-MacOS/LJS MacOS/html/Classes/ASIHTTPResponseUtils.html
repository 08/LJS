Overview

ASIHTTPResponseUtils is a class for extracting server response codes and error messages from an ASIHTTPRequest.

Parameters

aRequest

the request to query

Return Value

an NSInteger representing the response code.

Discussion

returns the response code of a request.

NOTE: ASIHTTPRequest appears to have problem with correctly identifying code 408. A status code 0 is converted to 408 by this method.
