pathFromOpenPanelWithPrompt:title:lastDirectory:fallBackDirectory:defaultsKeyOrNil:

creates a NSOpenPanel and returns the users selection as a path string.

+ (NSString *)pathFromOpenPanelWithPrompt:(NSString *)aPrompt title:(NSString *)aTitle lastDirectory:(NSString *)aLastDirectory fallBackDirectory:(NSString *)fallbackDirectory defaultsKeyOrNil:(NSString *)aDefaultsKeyOrNil

Parameters

aPrompt

the title of the select button – should be localized

aTitle

the title of the panel – should be localized

aLastDirectory

the last directory that was opened by nsopenpanel – if this directory exists, the panel will open to this directory

fallbackDirectory

if the last directory does not exist, then this the open panel will open to this directory

aDefaultsKeyOrNil

iff non-nil will use this key to store the users selection in NSUserDefaults

Return Value

a string representing the path to the file or directory of the user’s selection

Discussion

creates a NSOpenPanel and returns the users selection as a path string.
