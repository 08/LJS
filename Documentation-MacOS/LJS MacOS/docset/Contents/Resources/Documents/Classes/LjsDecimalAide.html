Overview

NSDecimalNumber is a powerful tool for handling currency, statistics, and other floating point data. The class name and the methods are, in my opinion, overly verbose and tend to clutter code at the worse times – when you are doing sensitive currency calculations or implementing a complex confusing statistical algorithm. And who can remember how to compare two NSDecimalNumbers?

Enter LjsDecimalAide – which provides methods for creating NSDecimalNumbers from various numeric values, logical comparisons and rounding.
